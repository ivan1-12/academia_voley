Instrucciones para ejecutar Academia Voley en otra PC

Requisitos previos:
- Python 3.10 o superior instalado.
- MySQL instalado y accesible.
- Clonar el repositorio desde GitHub.

Pasos:

1. Clona el repositorio:
   git clone https://github.com/ivan1-12/academia_voley.git
   cd academia_voley

2. Crear y activar el entorno virtual:
   py -3.14 -m venv .venv
   .\.venv\Scripts\Activate.ps1

3. Instalar dependencias:
   pip install -r requirements.txt

4. Crear el archivo de configuración local desde la plantilla:
   copy .env.example .env

5. Revisar y ajustar los valores en `.env`:
   - SECRET_KEY: valor secreto seguro.
   - MYSQL_HOST: dirección de MySQL local (por ejemplo `127.0.0.1`).
   - MYSQL_PORT: puerto de MySQL (por ejemplo `3306` o `3307`).
   - MYSQL_USER: usuario de MySQL.
   - MYSQL_PASSWORD: contraseña de MySQL.
   - MYSQL_DB: nombre de la base de datos (`academia_voley`).
   - ADMIN_EMAIL / ADMIN_PASSWORD: credenciales iniciales de administrador.
   - FLASK_HOST / FLASK_PORT: valores opcionales para la ejecución.

6. Crear la base de datos y cargar la estructura:
   mysql -u <usuario> -p < mysql_init.sql

7. Opcional: compilar traducciones si haces cambios en los archivos de idiomas:
   .\.venv\Scripts\python.exe scripts\compile_translations.py

8. Ejecutar la aplicación:
   .\.venv\Scripts\python.exe app.py

9. Abrir la aplicación en el navegador:
   http://127.0.0.1:5000

Notas:
- No subas el archivo `.env` al repositorio.
- Si usas XAMPP en el puerto `3307`, ajusta `MYSQL_PORT=3307` en `.env`.
- Este proyecto no requiere Docker para correr en desarrollo local.
